# react-boilerplate-webpack-babel

Splitting webpack configuration for development and production.

- common : `./config/common.js`
- dev : `./config/dev.js`
- production: `./config/prod.js`

## Installation

```
$ yarn

or

$ npm install
```

## Run

- Dev mode

  ```
  $ yarn dev
  ```

- Production Build
  ```
  $ yarn build
  ```
